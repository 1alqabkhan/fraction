You know in python there is no datatype for fraction numbers.So I made for made fraction datatype.
You can input a num see it in fraction form and this datatype can perform arithmetic operation as well.

METHOD I USE FOR THIS:

To get in fraction form I use "__str__" magic method for it. To add two fraction form I use "__add__" magic method. To subtract two fraction form I use "__sub__" magic method. To multiply two fraction form I use "__mul__" magic method. To divide two fraction form I use "__truediv__" magic method. To simplify fraction form I use math module operation called "math.gcd()" to find common between numerator and denomenator.

OUTSIDE THE CLASS:

I use simple function like input, if ,else defining function.
